# Браузерная игра Side Scroll Shooter

### Об игре

Летаем драконом и отбивает врагом

### Информация

Игра была написана в рамках изучения курса на Udemy по Phaser.js

## Курс: https://www.udemy.com/course/phaser-3/

Установка приложения

```sh
npm install
```

Запуск приложения

```sh
npm run dev
```
